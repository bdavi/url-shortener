## Requirements
- Postgres 12
- Redis 5
- Ruby 2.7
- Node 14
- Yarn 1.22
- Heroku CLI

## Setup
Run the following:
```
> bin/setup
```

## Build
Run the following:
```
> bin/build
```

## Run
Run the following:
```
> heroku local
```

Application is available at `localhost:5000`.

Sidekiq console is available at `localhost:5000/sidekiq`.
